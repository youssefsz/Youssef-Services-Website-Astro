## Overview

This repository contains the source code for the Youssef Services Website Astro. The platform serves as a central hub to showcase and offer professional software development capabilities. It is designed to be fast, responsive, and visually engaging, reflecting the high quality of services provided.

## Technologies Used

This project leverages a modern tech stack to achieve high performance and maintainability:

- **Astro**: For building a fast, content-focused website with island architecture.
- **React**: Used for dynamic and interactive UI components.
- **Tailwind CSS**: Utilizing version 4 for a utility-first, responsive, and maintainable styling approach.
- **Framer Motion**: Implements smooth, complex animations and transitions.
- **Lucide React**: Provides valid, scalable SVG icons.
- **TypeScript**: Ensures type safety and code quality throughout the development process.

## Features

- **Responsive Design**: Fully optimized for all device sizes, from mobile phones to high-resolution desktops.
- **High Performance**: Built with Astro to deliver minimal JavaScript and fast load times.
- **Interactive UI**: Engaging user interface elements powered by React and Framer Motion.
- **SEO Optimized**: Includes best practices for search engine optimization to maximize visibility.
- **Modern Aesthetics**: A clean, dark-themed design with glassmorphism effects and sophisticated typography.

## Getting Started

To explore the code or run the project locally, follow the steps below.

### Prerequisites

Ensure you have the following installed on your machine:
- Node.js (v18 or higher recommended)
- pnpm (or npm/yarn)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/youssefsz/Youssef-Services-Website-Astro.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Youssef-Services-Website-Astro
   ```
3. Install dependencies:
   ```bash
   pnpm install
   ```

### Development

Start the local development server:
```bash
pnpm dev
```
The application will be available at `http://localhost:4321`.

### Build

To create a production build:
```bash
pnpm build
```
The output files will be generated in the `dist/` directory.
